Reglas del Encriptador de Texto

Reglas del Encriptador

Encriptador Las letras se reemplazan segun las siguientes reglas:

  • a-> ai
  • e-> enter
  • i -> imes
  • o -> ober
  • u -> ufat

Desenciptación:El proceso inverso para recuperar el texto original.

Lo que no se puede hacer

  • No se permiten letras en mayúsculas.
  • No se deben usar caracteres especiales o acentos.
